# 飞书机器人配置指南

本文档介绍如何在飞书开放平台配置机器人，获取 ClaudeTalk 所需的凭据和权限。

## 前置条件

- 拥有飞书企业管理员权限（用于审批应用权限）

## 配置步骤

### 1. 创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn)
2. 登录后，点击「创建企业自建应用」或「创建应用」
3. 填写应用信息：
   - **应用名称**：如 `ClaudeTalk`
   - **应用描述**：如 `Claude Code AI 助手`
4. 选择应用类型：**企业自建应用**
5. 点击「创建」

### 2. 获取 App ID 和 App Secret

1. 进入应用详情页
2. 点击左侧菜单「**凭证与基础信息**」
3. 记录以下信息：
   - **App ID**：格式如 `cli_xxxxxxxxxxxxxxxx`
   - **App Secret**：格式如 `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 3. 添加机器人能力（必需）

1. 点击左侧菜单「**应用能力**」
2. 找到「**机器人**」卡片
3. 点击「**添加**」或「**启用**」
4. 确认启用

> **说明**：机器人能力是接收消息事件、使用 WebSocket 长连接、群聊 @ 识别的前提。

### 4. 开启长连接接收事件（必需）

1. 点击左侧菜单「**事件与回调**」
2. 找到「**使用长连接接收事件**」，点击「**开启**」
3. 确认开启

> **说明**：这是 WebSocket 长连接功能，必须开启才能接收消息。与 Webhook 方式不同，长连接不需要公网 IP。

### 5. 申请消息权限

1. 点击左侧菜单「**权限管理**」
2. 搜索并申请以下权限：

| 权限名称 | 权限代码 | 说明 |
|---------|---------|------|
| 获取与发送单聊、群组消息 | `im:message` | 发送和接收消息 |
| 获取与更新群组信息 | `im:chat` | 获取群聊信息 |
| 获取用户基本信息 | `contact:user.base:readonly` | 获取用户信息 |
| 查看群成员 | `im:chat.members:read` | 查看群成员列表 |

3. 点击「**申请权限**」
4. 填写申请理由：`用于 Claude Code AI 助手与用户交互`
5. 等待管理员审批

### 6. 发布应用

1. 点击左侧菜单「**版本管理与发布**」
2. 点击「**创建版本**」
3. 填写版本号和更新说明
4. 点击「**发布**」
5. 应用发布后才能正式使用

### 7. 将机器人添加到群聊（可选）

1. 在飞书中打开目标群聊
2. 点击群聊右上角「...」→「设置」→「群机器人」
3. 点击「添加机器人」
4. 搜索并添加你创建的应用

## 获取用户的 open_id

如需配置白名单，需要获取用户的 `open_id`：

1. 在飞书中打开用户资料
2. 点击「复制用户 ID」
3. 复制的 ID 即为 `open_id`

## 下一步

配置完成后，请参考 [ClaudeTalk 主文档](README.md) 进行以下操作：

1. 在项目目录下运行 `claudetalk --setup` 配置凭据
2. 输入飞书的 App ID 和 App Secret
3. 运行 `claudetalk` 启动机器人
